Main method to start the Nutrition Facts Detector System. 

This method initializes the database schema using the Db\+Init class, then enters a loop to present a menu-\/driven interface for interacting with the system. The user can choose various options such as adding, reading, updating, listing, and deleting nutrition facts. The loop continues until the user chooses to exit the system.
